# Business Analyst

Automatic business artifact extraction from source code.

## Installation

```bash
/plugin marketplace add yourname/business-analyst-marketplace
/plugin install business-analyst@business-analyst-marketplace
```

## Usage

Open any codebase and ask Claude to analyze it. The plugin automatically extracts:
- Business Rules
- Process Flows
- Data Specs
- User Stories
- Security/NFRs
- Integrations

Output saved to `docs/output/`
